React 组件集成指南

本指南将帮助您将现有的 React 组件集成到您的项目中。我们将使用 shadcn 项目结构、Tailwind CSS 和 TypeScript。

1. 项目设置

首先，确保您的项目支持 shadcn 结构、Tailwind CSS 和 TypeScript。如果还没有设置，请按照以下步骤操作：

# 创建新的 Next.js 项目
npx create-next-app@latest my-app --typescript --tailwind --eslint

# 进入项目目录
cd my-app

# 安装 shadcn UI
npx shadcn-ui@latest init

# 选择默认设置即可

2. 安装依赖

安装组件所需的所有依赖：

npm install @splinetool/runtime @splinetool/react-spline framer-motion

3. 创建组件目录

确保您的项目中有 /components/ui 目录。如果没有，请创建它：

mkdir -p components/ui

使用 /components/ui 目录非常重要，因为：

  • 它遵循 shadcn UI 的标准目录结构
  • 保持项目结构一致性，便于维护
  • 确保导入路径正确

4. 创建 utils 目录

确保您有 lib/utils.ts 文件，用于常用工具函数：

mkdir -p lib

在 lib/utils.ts 中添加：

import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"
 
export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

5. 添加组件文件

现在，将提供的组件复制到相应的文件中：

复制 splite.tsx 到 /components/ui/splite.tsx

复制 demo.tsx 到 /components/ui/demo.tsx

复制 Spotlight 组件到 /components/ui/spotlight.tsx

复制 Card 组件到 /components/ui/card.tsx

7. 组件分析

组件结构分析：

  • SplineScene：负责加载和显示 3D 场景
  • Spotlight：创建聚光灯效果
  • Card：提供卡片布局容器
  • SplineSceneBasic：组合以上组件创建完整演示

数据传递：

  • SplineScene 接收 scene URL 和可选的 className
  • Spotlight 接收位置和样式相关的 props

响应式行为：

组件使用 flex 布局和 Tailwind 响应式类（如 md:）确保在不同屏幕尺寸上正确显示

8. 最佳实践

  • 使用 Suspense 和 lazy 加载优化性能
  • 提供加载状态指示器
  • 应用统一的设计语言
  • 确保组件的可复用性
